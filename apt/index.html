Cloudera APT Repository

Contents

The Cloudera APT Repository provides a simple way to download and install Cloudera's distribution for Hadoop on Debian-based systems. The repository currently includes the Cloudera distribution for the following packages:
  • Hadoop
  • Pig
  • Hive
Browse the repository

Platform Support

We currently build and test our packages on the following Debian-based platforms:
  • Ubuntu 8.10 (intrepid) (x86 and amd64)
  • Ubuntu 8.04 (hardy) (x86 and amd64)
  • Debian 5.0 (lenny) (x86 and amd64)
  • Debian 4.0 (etch) (x86 and amd64)

If you are running on another architecture, you can still use the Cloudera Distribution for Hadoop - you will simply miss out on some performance enhancements and C libraries.

If you would like to see Debian packages built for other architectures or distributions, please let us know.

Instructions

Using Cloudera's APT Repository is very simple:
  1. Add repository. Create a new file /etc/apt/sources.list.d/cloudera.list with the following contents, taking care to replace DISTRO with the name of your distribution (find out by running lsb_release -c) 
    deb http://archive.cloudera.com/debian DISTRO contrib
deb-src http://archive.cloudera.com/debian DISTRO contrib
  2. Add repository key. (optional) Add the Cloudera Public GPG Key to your repository by executing the following command: 
    curl -s http://archive.cloudera.com/debian/archive.key | sudo apt-key add -
    This allows you to verify that you are downloading genuine packages.
  3. Update APT package index. Simply run: 
    sudo apt-get update
  4. Find and install packages. You may now find and install packages from the Cloudera repository using your favorite APT package manager (e.g apt-get, aptitude, or dselect). For example: 
    apt-cache search hadoop
sudo apt-get install hadoop
  5. For more detailed instructions and next steps, check out the tutorial.
