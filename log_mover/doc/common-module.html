get_key_cols(dict)

source code 

takes a dict and returns a string to be added to an "insert" statement, where each item is a key column name

Replaces:

  • '-' with '_'
  • ' ' with '_' and makes everything lowercase
Parameters:
  • dict (dictionary) - the dictionary whose keys will be returned in preparation for a SQL INSERT statement
Returns: string
a comma-delimited string, where each item is a SQL-compliant key name

get_values(dict)

source code 

takes a dict and returns a string to be added to an "insert" statement, where each item is an escaped value. Values are enclosed by single quotes (')

Replaces:

  • "'" with "'"
Parameters:
  • dict (dictionary) - the dictionary whose values will be returned in preparation for a SQL INSERT statement
Returns: string
a comma-delimited string, where each item is a SQL-compliant value

get_threshold_date(seconds)

source code 

Given a number of seconds, returns a date that is today - seconds. The date is in SQL format.

Parameters:
  • seconds (number) - the number of seconds, or the threshold
Returns: string (MySQL datetime format)
a string that is now - seconds

store_orm(table, dict, conn)

source code 

Given statitistics, store them in to MySQL with an ORM mechanism

Will convert keys to MySQL-compliant column names. See get_key_cols(dict)

Parameters:
  • table (string) - the SQL table
  • dict (dictionary) - the dictionary to insert
  • conn (connection) - a DB connection
